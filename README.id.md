**lvminance** adalah editor foto berbasis web yang dikembangkan sebagai proyek sampingan pribadi.

> **Catatan:** Repository ini digunakan secara eksklusif untuk laporan bug dan permintaan fitur. Source code aplikasi utama tidak dihosting di sini.

---

## Tentang Proyek Ini

Proyek ini lahir dari rasa penasaran untuk memahami bagaimana aplikasi editor foto bekerja dan apakah bisa diimplementasikan secara portable melalui browser. lvminance adalah eksplorasi eksperimental dalam membangun editor gambar fungsional yang berjalan sepenuhnya di browser, tanpa memerlukan instalasi atau pengaturan yang rumit.

---

## Pemberitahuan Penting

lvminance dikelola dalam waktu pribadi yang terbatas. Sebagai hasilnya:
- Respon dan perbaikan mungkin memerlukan waktu
- Tidak semua permintaan dapat segera ditangani
- Fitur dan UI dapat berubah tanpa pemberitahuan sebelumnya
- Tool ini bersifat eksperimental dan tidak ditujukan untuk penggunaan produksi atau mission-critical
